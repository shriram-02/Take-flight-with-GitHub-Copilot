# 🎮 Games in Python

## 🎯 Objective

In this assignment, students will learn how to create simple games using Python. The exercises focus on problem-solving, user interaction, loops, conditionals, and functions while building fun and interactive programs.

---

## 📝 Tasks

### ✅ Task 1: Create a Number Guessing Game

#### Description

Build a Python program where the computer randomly selects a number and the player tries to guess it.

#### Requirements

- Generate a random number using Python's `random` module
- Allow the player to input guesses
- Display hints such as:
  - "Too High"
  - "Too Low"
- Continue the game until the correct number is guessed
- Display the total number of attempts

#### Example

```python
Enter your guess: 5
Too Low

Enter your guess: 8
Too High

Enter your guess: 7
Correct! You guessed the number in 3 attempts.
```

### ✅ Task 2: Build a Rock Paper Scissors Game

#### Description

Create a Rock Paper Scissors game where the player competes against the computer.

#### Requirements

- Accept user input:
  - Rock
  - Paper
  - Scissors
- Randomly generate the computer's choice
- Determine the winner using conditional statements
- Display both choices and the result
- Allow the player to play multiple rounds

#### Example

```python
Player: Rock
Computer: Scissors

Result: You Win!
```

### ✅ Task 3: Create a Simple Quiz Game

#### Description

Develop a quiz game that asks the user multiple-choice questions and tracks the score.

#### Requirements

- Include at least 3 quiz questions
- Accept user answers from the keyboard
- Check whether answers are correct
- Keep track of the score
- Display the final score at the end

#### Example

```python
Question:
What is the capital of France?

a) Berlin
b) Madrid
c) Paris
d) Rome

Answer: c

Correct!
```
